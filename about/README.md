# 介绍

为什么使用 Hydro ？

- 安全：使用 cgroup 进行隔离，杜绝卡评测；
- 高效：Hydro 使用了沙箱复用技术，拥有极高的评测效率；
- 扩展：Hydro 支持安装额外模块进行扩展；
- 强大：配合 Judge 模块（或 HydroJudge 独立评测机），可支持 spj，交互题，提交答案题，文件IO 等多种特性；
- 简便：提供了多种迁移脚本，可从 Vijos 等快速升级至 Hydro；
- 自定：所有权限节点均可自由设置；
- 社区：Hydro 正在持续维护中；
