## 客观题

由于客观题的题面配置与测试数据配置均与其他题目不同，所以这里单独给出配置客观题的方法。

### 题面

题面需要以 yaml 格式给出。

```yaml
# 第 0 道题目
- desc: 这是一道普普通通的选择题
  choices: # 如果是选择题需要包含 choise 选项并列出题目的每一个选项的具体信息。
  - A. 1 + 1 = 1
  - B. 1 + 1 = 2
  - C. 1 + 1 = 3
# 第 1 道题目
- desc: 1 + 1 = ?
# 若不包含 choise 选项则该题为填空题
```

### 测试数据

仅需要配置 config.yaml 即可，不需要上传其他文件。

```yaml
type: objective # 表明该题为客观题
outputs: # 列举出每一题的正确选项与对应的得分
  - [B. 1 + 1 = 2, 50]
  - ['2', 50]
```
